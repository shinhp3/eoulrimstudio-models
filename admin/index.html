STL 모델 · GitHub Pages

이 페이지는 정적 안내·목록용입니다. 웹에서 STL을 올리려면 Cloudflare Worker 관리자(비밀번호)를 사용하세요. 루트 주소만 열면 3D 뷰어가 열립니다.

배포 방법

  • GitHub 저장소 Settings → Pages에서 소스를 main 브랜치 루트(또는 사용 중인 브랜치)로 설정합니다.
  • models/ 아래에 .stl 파일을 넣고 푸시합니다. 파일명은 영문·숫자·. _ - 만 권장합니다.
  • 뷰어 주소: 사이트 루트에 ?model=파일이름 (.stl 제외) 를 붙입니다.
